A continuación se muestra un grupo de palabras reservadas que le ayudaran a reemplazarlas por información y a generar codigo con GeneradorC

GAF(NombreTabla) > Esta palabra se reemplaza por el nombre de la tabla

GAFForEach > Sirve para repetir una intruccion el numero de veces equivalente a la cantidad de columnas

GAFEndForEach > Sirve para terminar la instrucción GAFForEach

GAFIfInt32 > Sirve para remplazar texto solo si el campo es entero

GAFIfString >Sirve para remplazar texto solo si el campo es cadena

GAFIfDateTime > Sirve para remplazar texto solo si el campo es fecha

GAFIfBoolean > Sirve para remplazar texto solo si el campo es boleano

## > Sirve para poner comentarios dentro del archivo

GAF(ID) >Sirve para obtener el id de la tabla

GAF(,) > Sirve para poner una coma al final de una linea excepto en el ultimo campo de la tabla

GAF(*)

>El asterisco lo podemos reemplazar por cualquier tipo de campo que tiene una tabla
>por ejemplo GAF(ColumnName) que da el nombre de columna,GAF(DataType) que da el tipo de dato
>para camelCase agregamos el las letras cC, ejemplo GAFcC(ColumnName)

Nombre y extensión del archivo

##En el segundo renglon va el nombre del archivo, en el tercero va el nombre de la extensión

GAF(IdDataTypeName)

Funciona para reemplazar el id con su tipo de dato por ejemplo @Ciud_ID int GAF(ID) GAF(IdDataTypeName)

GAFForSinIdEach

Sirve para repetir una instrucción el número de veces equivalente a la cantidad de columnas excepto el id de la tabla.

GAFEndForSinIdEach

Sirve para terminar la instrucción GAFForSinIdEach

GAF(Date) Sirve para traer la fecha actual
